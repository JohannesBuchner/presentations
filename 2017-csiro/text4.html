Column density NH

  • NH … number of H atoms along line-of-sight
  • NH = few atoms / cm³ x few kpc = 10²²/cm²
  • Compton-thick: ~ 1024 cm²
  • O, Fe, He
    • absorb soft X-rays via photo-electric effect
    • Measure NH assuming local abundances from X-ray spectrum
