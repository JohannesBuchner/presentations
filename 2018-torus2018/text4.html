XARS

  • X-ray Absorption Re-emission Scattering
  • Ray-tracing code for X-rays
  • Open source, Python
  • Arbitrary geometries:
    • Torus, Sphere,
    • Millions of spheres
    • Grids (like from hydrodynamic simulations)
