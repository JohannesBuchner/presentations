Aims

Requires

  1. need good method to recover properties from limited information (low-count spectra)
  2. need good method to recover population from biased sample

Analysis of low-count spectra

  • Parameter estimation
    • Maximum likelihood - C-stat optimization, contours
    • Bayesian - C-stat as likelihood, but full parameter space exploration
  • Model selection
    • likelihood ratio based - limited to nested models, no borders
    • Bayesian - no such limitations
  • Outcome: for each spectrum in CDFS,
    • parameter probability distribution
    • (strength of) evidence for each tested model (see next slide)

Population properties from sample statistics

  • Every sample is biased, this one in particular
  • We know how the bias works - sensitive area
  • Drawing inference on the population

Status of the field

  • Maximum likelihood estimators with bins
  • "Bayesian" inference on ad-hoc parametric models
  • Priors not motivated, essentially doing maximum likelihood style parameter estimation, but with Monte-Carlo

New Idea: field reconstruction

  • non-parametric field reconstruction
  • Bayesian inference -- parametric model over fine grid
  • prior that neighboring bins are similar (preference to be equal, power laws)
  • determine this prior from data at the same time

New Idea: field reconstruction

  • Bins: variables
  • +3 "correlation strength" parameters
  • just optimize and do MCMC

What's the point?

  • No binning before analysis
  • Discover a model and its correlations instead of imposing it
